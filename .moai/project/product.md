# Deskflow — 제품 문서

## 제품 개요

Deskflow는 개인 생산성을 극대화하는 올인원 데스크톱 허브 애플리케이션입니다.
Electron 기반으로 Windows, macOS, Linux에서 동작하며, React + TypeScript로 구현된 모던 UI를 제공합니다.

## 핵심 가치

- **통합 작업 공간**: 북마크, 할 일, 메모, 검색을 하나의 화면에서 관리
- **개인화**: 카테고리와 링크를 자유롭게 편집하는 완전한 커스터마이징
- **영속성**: 앱 재시작 후에도 모든 데이터 유지 (electron-store 기반)
- **접근성**: 다크/라이트 테마, 한국어 날짜, 직관적 UX

## 대상 사용자

- 생산성 향상을 원하는 개인 사용자
- 자주 방문하는 사이트를 빠르게 접근하고 싶은 개발자
- 할 일과 메모를 한 곳에서 관리하고 싶은 직장인

## 핵심 기능

### 위젯 모음
| 위젯 | 설명 |
|------|------|
| 실시간 시계 | 한국어 날짜·요일 포함, 시간대별 인사말 |
| Google 검색바 | 새 탭에서 Google 검색 실행 |
| 즐겨찾기 그리드 | 카테고리별 링크 관리 (기본: Work, Dev, Media, Tools) |
| 할 일 목록 | 추가/완료/삭제, 남은 항목 수 표시 |
| 빠른 메모 | 자동 저장, 줄바꿈 지원 |

### 커스터마이징
- 카테고리 추가/편집/삭제 (⚙️ 버튼)
- 링크 아이콘(이모지), 이름, URL 자유 편집
- 다크/라이트 테마 전환
- 모든 데이터 자동 저장 (세션 간 유지)

### 고급 기능

- **RAG 시맨틱 검색** (v0.8.0+, SPEC-SEARCH-RAG-001): 로컬 Ollama 기반 자연어 시맨틱 검색. Command Palette에서 4자 이상 쿼리 입력 시 북마크를 의미론적 유사도로 검색하며, Ollama 미설치 시 자동 폴백

## 기술 개요

### 코어 스택
- **런타임**: Electron 35.x (크로스플랫폼 데스크톱)
- **UI**: React 19 + TypeScript 5.7 (strict mode)
- **빌드**: electron-vite 5.0.0 + Vite 7.x (3-프로세스 구조)
- **스토리지**: electron-store 10.x (IPC 브릿지)
- **상태**: Zustand 5.x
- **패키징**: electron-builder 25.x

### 개발 도구
- **테스트**: Vitest 3.x + @testing-library/react 16.x
- **린터**: ESLint 9 (flat config) + typescript-eslint
- **포매터**: Prettier 3.x
- **타입 검사**: TypeScript 5.7 (Project References)
