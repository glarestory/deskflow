## 사전 준비

### 1. Ollama 설치

**macOS / Linux**
```bash
curl -fsSL https://ollama.com/install.sh | sh
ollama serve &   # 백그라운드 실행 (또는 서비스로 등록)
```

**Windows**
https://ollama.com/download/windows 에서 설치 후 실행.

### 2. 임베딩 모델 다운로드

```bash
ollama pull nomic-embed-text
# 약 274MB, 나오미-임베딩-텍스트 v1.5
```

### 3. CORS 설정 (웹 빌드에서만)

```bash
# 개발용 (모든 오리진 허용 - 로컬 전용)
export OLLAMA_ORIGINS="*"

# 또는 특정 오리진만
export OLLAMA_ORIGINS="http://localhost:5173"

# Windows PowerShell
$env:OLLAMA_ORIGINS="*"

# 설정 후 Ollama 재시작 필요
killall ollama && ollama serve &
```

### 4. 앱 실행

```bash
# Electron
npm run dev

# Web
npm run dev:web
```
